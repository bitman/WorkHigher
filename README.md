# Freedom Flow Fitness

A weekly exercise plan for elderly adults, designed to improve mobility, strength, and balance.

## About

This website provides a comprehensive exercise program tailored for elderly adults, featuring:
- Weekly structured exercise routines
- Visual illustrations for each exercise
- Easy-to-follow instructions
- Focus on safety and accessibility

## Deployment

This site is automatically deployed to GitHub Pages. See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed deployment instructions.

**Live Site**: https://bitman.github.io/WorkHigher/

## Local Development

To preview the site locally:

1. Open `index.html` in a web browser, or
2. Run a local web server:
   ```bash
   python -m http.server 8000
   ```
   Then visit http://localhost:8000
